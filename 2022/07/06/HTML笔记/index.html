1.HTML笔记

1.初始HTML

1.1 HTML是什么？

​ HTML是用来描述网页的一种语言

​ HyperText MarkUp Language 简称HTML

​ HTML不是一种编程语言 而是一种标记语言

​ HTML使用标记标签来描述网页

​ HTML文档也叫web页面

后缀

​ HTML的后缀 .html .htm

1.2.实例分析

​ HTML代码 

<!DOCTYPE html> //声明为HTML5文档
<html lang="en"> //元素是HTML页面的根元素
<head>  //包含了文档的元数据（meta）数据，
    <meta charset="UTF-8"> //页面的编码格式
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>元素包含了可见的页面内容
    <h1>我是一个标题</h1>   //这个属于是可见页面的内容
    <h1>我也是一个标题</h1>
    <p>我是一个段落</p>
</body>
</html>

3.5HTML元素语法

​ 1.元素语法：

​

元素

  • html元素以开始标签起始
  • html元素以结束标签终止
  • 元素的内容是开始标签与结束标签之间的内容
  • 某些html元素具有空内容（empty content）
  • 空元素可以在开始标签在中进行关闭（以开始标签的结束而结束）
  • 大多数html元素可拥有属性
  • 大多数html元素可以嵌套（html元素可以包含其他html元素）html文档由相互嵌套的html元素组成

​ 2.html空元素

  • 没有内容的html元素被称为空元素 空元素是开始标签中关闭的
  • br标签就是没有关闭标签的空元素（换行的）
  • 在xhtml,xml以及未来的版本中的html中所有元素必须被关闭
  • 在开始标签中必须添加斜杠，比如br就是关闭空元素的正确方法，xhtml和xml都接收这种方式
  • 即使br在所有的浏览器中都是生效的但使用br/其实是更长远的保障

​ 3.html提示：使用小写标签

  • html标签对大小写不敏感：

    等同于

    ，许多网站都使用大写的html标签但是我一般使用的都为小写，，因为w3c在html中推荐使用小写所有在xhtml版本中强制使用小写。

3.6路径

3.6.1绝对路径

​ 定义：从根目录开始，是一个完整的路径

​ eg D:\demo1\img\good.jpg

​ 比如：德云社的绝对路径就是北极市西城区前门大栅栏大街38号

3.6.2 绝对路径

​ 定义：项目目录的路径，在开发中引用js,css开发者图片等都用绝对路径

​ eg：…/img/good.jpg

​ 出门右拐第二个屋子就是德云社

​

3.6.3两者之间的区别：

​ 符号 含义 例子

​ . 当前所在的目录

​ ../ 上一层目录

​ ../../ 上一层目录的上一层目录

​ / 代表根目录

D:/abc 绝对路径

3.7HTML超链接

​ 1.定义：HTML使用超级链接 与网络上的林外一个文档相连，几乎可以在网页中找到链接，点击链接可以从一个一张页面跳转到另外一张页面

​ HTML使用标签 来设置文本链接

1.1超链接可以是一个字，一个词，也可以是一副图片，你可以点击这些内容来跳转到新的文档或者当前文档的某个部分 当年把鼠标指针移入到页面的某个超链接上时候，鼠标箭头就会变成一个小手

在标签中使用了href属性来描述链接地址

1.2默认情况下，链接会出现下面几个方式出现在浏览器中

  • 一个未访问过的超链接显示为蓝色字体，并且带有下划线
  • 访问过的链接显示紫色并带有下划线
  • 点击链接时候，链接显示红色并带有下划线

一般在正常的网站都会通过CSS对超链接进行修饰。

4.属性

4.1属性的定义

  • HTML元素可以设置属性
  • 属性可以在元素中添加附加信息
  • 属性一般描述于开始标签
  • 属性总是以名称/键值对的形式出现
  • 比如： name=” value “

4.2 属性实列

​ HTML链接由标签定义，链接的地址在href属性中指定

​ href代表属性，href后面写的值 叫做属性值

[]: 跳转到百度页面

4.2.1HTML属性常用应用属性值

属性值应该始终包裹在引号内

双引号是最常见的，不过使用单引号也没有问题

tip：在某个别的情况下，比如属性本身就包含了引号，那么咱们就必须使用单引号 eg name=’John “ShotGun”Neison’

口诀：外单内双 外双内单

常用属性

class 为html元素定义一个或者多个类名（classname）

id 定义元素的唯一ID

style 规定了元素的行内样式

title 描述了元素的额外信息

  • 属性和属性值尽量小写 这样可以为未来做准备
  • class的名称可以被重复的使用
  • id的名称只能出现一次
